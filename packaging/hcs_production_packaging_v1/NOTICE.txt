Harmonic Conductor Studio — Production Packaging v1 Notice

This release path packages the Harmonic Conductor Studio application only. It must not package local user custody data, local SQLite libraries, private Identity Vault material, exported HFIELD bundles, proof files, or development scratch artifacts.
