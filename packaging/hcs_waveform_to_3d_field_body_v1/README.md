# HCS Waveform to 3D Field Body v1

Contract: `aiweb.hfield.waveform_to_3d_field_body.v1`

This patch turns the bubble concept into a deterministic rendering rule.

Each track’s notes are treated as a waveform/envelope source. HCS extracts timing, duration, velocity, pitch range, RMS/peak energy, and envelope samples. Those values produce an elongated resonant field body placed into the Glass Reader plane.

This is not random sphere art. It is a score-derived waveform body.
