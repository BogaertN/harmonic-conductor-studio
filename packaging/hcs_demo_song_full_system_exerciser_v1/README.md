# HCS_DEMO_SONG_FULL_SYSTEM_EXERCISER_V1

Adds a long Rust-owned deterministic demo composition for Harmonic Conductor Studio.

This preset is intended to exercise the complete stack:

- Persistent score spine
- Composer piano roll
- Composer waveform editor true sound body contract
- deterministic music playback
- conductor gesture timeline
- Glass Reader 3D sound-body extrusion
- cymatic/field/playhead response

The normal user action is **Load Full Demo Song**. The preset ID is `full_system_exerciser_v1`.

The demo is 32 bars at 96 BPM and uses all nine conductor gesture IDs across primary and expressive hand tracks.
