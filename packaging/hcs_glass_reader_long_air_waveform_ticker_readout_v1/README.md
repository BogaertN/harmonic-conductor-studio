# HCS_GLASS_READER_LONG_AIR_WAVEFORM_TICKER_READOUT_V1

This patch corrects the Glass Reader production view after the air waveform / piezo ticker pass.

It makes the waveform bodies much longer, larger, and more inspectable while keeping them floating in air. It removes always-on body labels and gates note/frequency readouts to the glass timing plane only. The PMUT-style cymatic rings now appear only at active glass crossings.
